SIMONO SEO FIX

1. Copy sitemap.xml, robots.txt, and vercel.json to the root of your simono-website project.
2. Replace the existing files when asked.
3. Commit and push to GitHub.
4. Wait for Vercel to deploy.
5. In Google Search Console, submit the FULL URL:
   https://simono.io/sitemap.xml

Important: Browsers often display an XML sitemap as plain text when no stylesheet is attached. That is normal and does not mean the XML is invalid.
